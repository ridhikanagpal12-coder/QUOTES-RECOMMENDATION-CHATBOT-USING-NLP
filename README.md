QuoteBot

A conversational chatbot that recommends quotes based on a user's mood or request.
The system uses Natural Language Processing (NLP) to understand user intent and deliver relevant quotes such as motivational, love, or humorous quotes.

The chatbot is built using Rasa, an open-source framework for building contextual AI assistants.

Features

1) NLP-based intent detection

2) Quote recommendations based on mood or request

3) Multiple quote categories:

    Motivational

    Love

    Humor

4) Conversational dialogue handling using Rasa

5) Custom actions for dynamic responses

6) Simple frontend interface for user interaction


Technologies Used:

1) Python

2) Rasa Framework

3) Natural Language Processing (NLP)

4) HTML / CSS / JavaScript

5) Machine Learning
